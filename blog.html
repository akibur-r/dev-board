Question-1: What are the different ways to select an element in the DOM?

  1. Using querySelector

    An element can be selected using .querySelector() method. For example, to select the element with 'test' id, the syntax will be: document.querySelector('#test')This returns only one element matching the provided query.

  2. Using querySelectorAll

    An element can be selected using .querySelectorAll() method. For example, to select the elements with 'test' class, the syntax will be: document.querySelectorAll('.test')This returns a list of the element matching the provided query in NodeList format, which is an array-like object.

  3. Using getElementById

    An element can be selected using .getElementById() method. For example, to select the element with 'test' id, the syntax will be: document.getElementById('test'). This returns only one element matching the provided id.

  4. Using getElementsByClassName

    An element can be selected using .getElementsByClassName() method. For example, to select the element with 'test' class, the syntax will be: document.getElementsByClassName('test'). This returns a collection of HTML elements matching the provided class name, which is an array-like object.

  5. Using getElementsByTagName

    An element can be selected using .getElementsByTagName() method. For example, to select the p tags, the syntax will be: document.getElementsByTagName('p'). This returns a collection of HTML elements matching the provided tag name, which is an array-like object.

  6. Using getElementsByTagNameNS

    An element can be selected using .getElementsByTagNameNS() method. For example, to select the p tags within http://www.w3.org/2000/svg namespace, the syntax will be: document.getElementsByTagNameNS("http://www.w3.org/2000/svg", "p"). This returns a collection of HTML elements matching the provided tag name within the namespace, which is an array-like object.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. innerHTML returns the whole element including the tag, comment, text, class, id and all other attributes.
  2. innerText returns the visible and explicit texts inside an element only.
  3. textContent returns all the texts inside an element.

Question-3: What is event delegation in the DOM?

When we add a single event listener to the parent element instead of adding multiple event listeners to the child elements, that's called event delegation. We can take advantage of event bubbling and efficiently handle the children when an event occurs.

Question-4: What is event bubbling in the DOM?

When we trigger an event on an element, the event listener digs through all the elements from the root in the hierarchy to find the actual target of the event. When the target is found, the parents are dropped and the target is caught. It's called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Creating Element

    We can create an element using JavaScript by using the createElement function. The syntax to create a paragraph element is: document.createElement('p')

  • Adding Element

    We have to target the parent element. Then we have to use the appendChild method to add an element. The syntax to add the created paragraph element to the body element is: document.getElementByTagName('body').appendChild(document.createElement('p'))

  • Removing Element

    We have to target the parent element. Then we have to use the removeChild method to add an element. The syntax to remove the added paragraph element to the body element is: document.getElementByTagName('body').removeChild(document.getElementByTagName('p'))
